# 🤖 BotBuddy — AI Chatbot SaaS

A SaaS platform that allows small businesses to create AI-powered customer support chatbots in minutes — no coding required!

## 🌍 Live Demo
👉 [API Live](https://web-production-e9d55.up.railway.app)
👉 [API Docs](https://web-production-e9d55.up.railway.app/docs)

## ✨ Features
- 🔐 User Authentication (Signup & Login)
- 🤖 AI Chatbot Creation
- 💬 Live Chat Widget
- 📊 Chat History
- 🌍 Embeddable on any website

## 🛠️ Tech Stack
- **Backend:** Python, FastAPI
- **Database:** SQLite
- **AI:** Together AI / Gemini
- **Hosting:** Railway
- **Frontend:** HTML, CSS, JavaScript

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/ZaiNiiiMuGHaL/chatbot-saas.git
cd chatbot-saas
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Create .env file


TOGETHER_API_KEY=your_key_here


### 4. Run the server
```bash
uvicorn main:app --reload
```

### 5. Open in browser

http://127.0.0.1:8000
